최소 2문항, 최대 8문항 — 1~2분이면 끝납니다.

Q1
AI 에이전트가 뭔지 아시나요?
단순 AI 챗봇이 아니라, 도구를 사용하고 스스로 판단해서 작업을 수행하는 AI를 말합니다.
처음 듣는 말이에요
들어는 봤는데 정확히는 모르겠어요
네, 알고 있어요
Q2
에이전트가 "도구(Tool)"를 사용한다는 게 어떤 의미인지 아시나요?
예: 파일을 읽거나, 웹 검색을 하거나, 코드를 실행하는 등의 행위를 AI가 직접 호출하는 것
잘 모르겠어요
아, 그런 뜻이군요 (지금 알았음)
네, 이미 알고 있었어요
Q3
클로드 코드(Claude Code)를 써보셨나요?
터미널(CLI)에서 실행하는 AI 코딩 에이전트입니다.
뭔지 모르겠어요
들어는 봤는데 안 써봤어요
클코는 안 써봤지만 다른 AI 코딩 도구는 써봤어요
써본 적 있어요
현재 사용 중이에요
Q4
도구를 구분해서 쓰고 계신가요?
예: 파일 읽기 → Read, 검색 → Grep, 터미널 명령 → Bash 등
구분 없이 AI가 알아서 쓰는 대로 써요
대략 알지만 의식하진 않아요
네, 상황에 맞게 골라 씁니다
Q5
권한 모드를 알고 계신가요?
자동허용 / 확인 후 실행 / 도구별 허용·거부 등 실행 권한을 제어하는 기능
몰랐어요
알긴 하는데 건드려본 적 없어요
설정해서 쓰고 있어요
Q6
CLAUDE.md 파일이 뭔지 아시나요?
프로젝트별 지시사항을 적어두는 설정 파일로, AI의 행동 규칙을 정의합니다.
처음 들어요
있다는 건 아는데 직접 작성해본 적 없어요
직접 작성해서 쓰고 있어요
Q7
다음 중 알고 있거나 사용해본 것을 모두 골라주세요.
하나도 모르면 맨 아래를 선택해주세요.
훅 (Hook) — 도구 실행 전후에 자동 실행되는 스크립트
MCP 서버 — 외부 도구/API를 클로드에 연결
스킬 (Skill) — 슬래시 명령으로 호출하는 재사용 프롬프트
커스텀 에이전트 — 특정 작업 전담 서브에이전트
하나도 모르겠어요
