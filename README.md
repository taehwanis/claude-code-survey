# 클로드 코드 이해도 셀프체크

클로드 코드(Claude Code)를 얼마나 알고 있는지 스스로 확인해보는 간단한 설문입니다.

**[지금 해보기 →](https://taehwanis.github.io/claude-code-survey/)**

## 어떤 설문인가요?

- 최소 2문항, 최대 8문항 — 1~2분이면 끝납니다
- 응답에 따라 다음 질문이 달라지는 분기 방식
- 끝나면 바로 등급과 요약을 확인할 수 있습니다

## 등급 체계

| 등급 | 설명 |
|------|------|
| 초급 | AI 에이전트 개념부터 시작하는 단계 |
| 중급 | 클로드 코드를 사용하고 있지만 커스텀은 아직 |
| 고급 | 훅, MCP, 스킬 등 확장 기능을 파악하고 있는 단계 |
| 전문가 | 직접 만들어서 워크플로우에 녹여 쓰는 단계 |

## 다루는 주제

- AI 에이전트 / 도구(Tool) 개념
- 클로드 코드 사용 경험
- 권한 모드, CLAUDE.md
- 훅, MCP 서버, 스킬, 커스텀 에이전트
